## Points

* 使用RxJava配合Retrofit2做网络请求
* 使用MVVM架构
* 使用RecyclerView实现下拉加载
* 使用RecyclerView多布局
* 自定义View
* 使用ijkplayer播放视频
* Android 4.4 及以上版本的状态栏适配
* 数据离线缓存
